# mochaccino

A Web Service that generates mock data for any use cases.

## Why Mochaccino
- Fast
- Simple to use

## Supported formats:
- JSON
- CSV
- *SQL* (in active development)

## Data providers:

- First name
- Last name
- Address
- Country
- City
